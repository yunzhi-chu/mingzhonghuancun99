## 一、色彩体系

## 三、Bento Grid 布局

### 3.1 网格系统

```
桌面端:         4 列 Bento Grid (gap: 16px)
平板端:         2 列
移动端:         1 列
```

### 3.2 卡片尺寸

| 模块 | 桌面列数 | 内容 |
|---|---|---|
| 综合评分 | 全宽 4 列 | 圆环仪表盘 + 诊断文字 |
| 命中率分析 | 2 列 | 核心 KPI 卡片组 |
| 节省预估 | 2 列 | 月省 + 节省比例 |
| Agent 列表 | 2 列 | 每个 agent 状态卡片 |
| 配置状态 | 2 列 | 表格形式 |
| 优化建议 | 全宽 4 列 | 建议列表 |

### 3.3 卡片规范

```
圆角:      12px
内边距:    24px (桌面) / 16px (移动)
边框:      1px solid rgba(255,255,255,0.06)
背景:      rgba(17,24,39,0.8)
背虚化:    backdrop-filter: blur(20px)
阴影:      0 4px 24px rgba(0,0,0,0.2)
悬浮效果:  边框亮度 +0.02, 阴影加深
```

## 四、组件风格

### 4.1 综合评分环

- 圆环直径: 160px
- 进度条宽度: 10px
- 颜色: 按语义色渐变（红色→琥珀→蓝色→绿色）
- 中心显示: 分数 + "/100"
- 环下方: 诊断文字

### 4.2 统计卡片

- 图标 + 数字 + 标签
- 数字使用等宽字体
- 微弱的底部边框区分

### 4.3 进度条 (命中率)

- 高度: 8px
- 圆角: 4px
- 颜色: 渐变，从左到右（紫→青）
- 背景色: 卡片底色
- 中间标注百分比

### 4.4 优化建议卡片

- 左侧 4px 颜色条（按严重程度）
- 标签：严重程度胶囊 + 分类标签
- 标题：粗体
- 详情/影响/修复：三行描述

### 4.5 标签/徽章 (Badge)

| 类型 | 颜色 | 背景 |
|---|---|---|
| critical | `#ef4444` | `rgba(239,68,68,0.15)` |
| warning | `#f59e0b` | `rgba(245,158,11,0.15)` |
| info | `#06b6d4` | `rgba(6,182,212,0.15)` |
| success | `#10b981` | `rgba(16,185,129,0.15)` |

## 六、交互行为

| 行为 | 效果 |
|---|---|
| 卡片 hover | 边框亮度提升，阴影加深 |
| 优化建议 hover | 左侧颜色条宽度增加至 6px |

| 建议展开/收起 | 可折叠细节 (details/summary) |
